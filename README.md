## What's this?
* 로그라이크 게임 [(참조)](https://ko.wikipedia.org/wiki/%EB%A1%9C%EA%B7%B8%EB%A5%98_%EA%B2%8C%EC%9E%84)
* RPG로 확장

## How Play it?

### 환경
* 우분투 리눅스 18.04.3
* python 3.6.8

### 실행법
* pygame 다운로드
 * pip3이 설치되어 있어야 합니다.
 * pip3 install pygame
     
* Play.py 실행 ( src에 들어가서 ./Play.py  실행이 되지 않을 시, chmod +x Play.py )

* 시작 메뉴 : 선택은 화살표로 해주세요
